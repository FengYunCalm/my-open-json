# Graphify Outputs

Typical outputs appear under `graphify-out/`.

## Common Artifacts

- `GRAPH_REPORT.md` - plain-language summary
- `graph.json` - machine-readable graph
- `graph.html` or equivalent HTML visualization
- optional exports such as `graph.svg`, `graph.graphml`, or `cypher.txt`

## What to Report Back

When graphify completes, summarize:
- what mode ran
- what path or corpus was used
- which key artifacts were produced or updated
- any warnings such as skipped files, large-corpus limits, or partial updates
