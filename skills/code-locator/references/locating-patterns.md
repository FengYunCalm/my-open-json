# Locating Patterns by Project Type

## Web / React / Next

Look first in:
- `src/`
- `app/`
- `pages/`
- `components/`
- `lib/` or `utils/`

## Backend / Services

Look first in:
- `src/api/`
- `src/routes/`
- `src/services/`
- `src/db/`
- `internal/`

## MUD / LPC

Look first in:
- `modules/`
- `daemons/`
- `cmds/`
- `inherit/`
- `system/`

## Mobile

Look first in:
- `app/`
- `pages/`
- `components/`
- `services/`
- `static/`

Use these as search hints, not as absolute rules.
